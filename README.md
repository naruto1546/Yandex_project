<h2> Пояснительная записка </h2>

<h4> Автор проекта - Шкаленко Леонид </h4>
<h4> Описание игры: Эта игра предназначена для развития или уледшения абстрактное мышление. Перед игроком находится коробка, в которой расположены металлические шары в разных секциях, шары превышают размер секции. Игрок может пустить определённое кол-во металлических шариков с разных сторон коробки, чтобы определить в каких секциях или секции находятся металлические шары. </h4>
<h4> Цель игры - в каждом уровне найти в какой секции находится центр каждого металлического шара. </h4>
